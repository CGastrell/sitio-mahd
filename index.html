ArgenMap

es un servicio web orientado a diseñadores y programadores web que te permite mostrar un mapa de Argentina en un sitio web con los datos del Instituto Geográfico Nacional de Argentina.

v.1. jquery + google maps api + argenmap.jquery
v.2. jquery + argenmap.jquery
ArgenWamp

es una herramienta que te permite construir mapas web con datos guardados en una Google Spreadsheet. Utilizando como mapa base Argenmap. El código de argenWAMP está disponible bajo una licencia libre.

GlobalWamp

es una herramienta que te permite construir mapas web con datos guardados en una Google Spreadsheet. Utilizando como mapa base OSM Open Street Map. El código de GlobalWAMP está disponible bajo una licencia libre.
